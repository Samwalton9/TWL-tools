Overview

The metrics data shown here is the total number of links to a URL pattern from the language Wikipedia denoted above each graph. The data is not limited to any subset of users.

This data allows The Wikipedia Library (TWL) team and its publisher partners to assess the degree to which TWL resources are used on Wikipedia and therefore how successful a partnership is.

Future

The existing link metrics data is limited in its usefulness. It must be actively collected on a regular basis, only has historical data as far back as we began tracking, doesn't distinguish between links added by editors with access through The Wikipedia Library or via other means, and can't provide any information about removed links.

We are working on improvements to Mediawiki's link tracking which will add a new data source tracking every new link addition or removal, the date it was added/removed, and the username of the editor who added/removed the link. This data will be displayed alongside the overall tracking graphs in the near future.

Technical details

Link metrics are collected from the Externallinks table of Toolforge's production database replicas. Until December 2017 this data was collected manually and sporadically. It is now collected automatically on a weekly basis through Toolforge.

The source code for collecting this data is viewable on GitHub.
